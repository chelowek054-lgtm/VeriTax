# Суть проекта

Создание модульной системы, которая превращает хаотичный поток сообщений из мессенджеров и форумов (Telegram, Discord, RSS) в верифицированную, графово-векторную базу знаний о моделях искусственного интеллекта (генерация изображений, текстов и т.д.).

## Основная цель

Предоставлять пользователю аргументированные ответы на нечеткие сравнительные запросы («Какая модель лучше для RTX 3090?») с обязательными ссылками на первоисточники и возможностью глубокого погружения в «досье» конкретной модели.

## Ключевые особенности архитектуры

- Защита от мусора и ошибок: Многоуровневая фильтрация, карантин для неподтвержденных данных и механизмы отзыва ошибочных фактов без их удаления (сохранение истории правок).
- Эйлерова логика пересечений: Данные не дублируются, а связываются через мета-узлы обсуждений, что позволяет строить сложные аналитические срезы.
- Административный факт-чекинг: Ручное управление таксономией и разрешение конфликтов мнений для поддержания высокого качества выдачи.
- Доказательность: Каждое утверждение в ответе бота подкреплено ссылкой на исходное сообщение или ветку обсуждения.

## Итоговый продукт

API и витрина данных, генерирующая «Вики-страницы» моделей на лету, где отображается не маркетинговая информация, а реальное агрегированное мнение сообщества с указанием уровня его достоверности.
